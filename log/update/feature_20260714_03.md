# v2.2.0 — 动效体系 Phase 2b + Phase 4.3 + Phase 5.1（仪式动画补全 + 路由转场差异化）

**类型**：feature
**日期**：2026-07-14 20:55
**版本**：2.2.0+18
**影响范围**：core/animation/ritual/、core/animation/transitions/、shared/widgets/、router/app_router.dart、features/home/home_page.dart、features/history/history_page.dart、features/settings/settings_page.dart、core/theme/animations.dart、pubspec.yaml、builds/

## 变更描述

承接 v2.1.0 的动效体系，落地 `docs/NEXT_PLAN/NEXT_PLAN_v2.1.0_ANIMATION.MD` 设计方案的 Phase 2b（后 4 套仪式动画）+ Phase 4.3（路由转场差异化）+ Phase 5.1（DivinationLoadingIndicator）。

10 术全部具备仪式入场动画（v2.1.0 完成 5 套 + 小六壬，本次补全剩余 4 套）；每术路由转场有专属视觉语言（小六壬太极旋、周易爻堆叠、紫微辐射、奇门九宫、抽签卷轴、测字字符浮现、大六壬双盘旋转、风水罗盘指针扫描、梅花数字合体、掷筊抛物线）；加载指示器全面中国风化（六边形外环 + 八卦点环 + 中心呼吸金光）。

按用户决策执行：测字简化为浮现+五行色染（不做笔顺分解）；仪式动画 3 秒后显示跳过按钮；时长 3-6s 标准版。

## 修改详情

### Phase 2b：后 4 套仪式动画

#### 2.1 梅花数字撞击（4s）
- [mobile/lib/core/animation/ritual/meihua_ritual.dart]（新增）
  - 5 阶段：环境金光渐显 + 两数字飘落 → 撞击爆发（16 颗金光粒子） → 数字消散成五行色光 + 卦象淡入 → 动爻从顶降临 → 金色描边
  - 复用 `features/meihua/algorithm/divine.dart` 的算法（先天八卦序 + 动爻计算）
  - `_MeihuaRitualState extends RitualAnimationState<MeihuaRitual>`

#### 2.2 掷筊翻转落地（3s）
- [mobile/lib/core/animation/ritual/jiaobei_ritual.dart]（新增）
  - 4 阶段：黑底渐显 → 杯筊抛物线下落 + 翻滚 2 周 → 落地弹跳 + squash 形变 → 结果定型（圣/笑/阴筊光晕）
  - 新月形杯筊用 `Path.quadraticBezierTo` 构成，旋转 π 翻转方向
  - `_RitualParams.random()` 随机生成两片正反
  - `_JiaobeiRitualState extends RitualAnimationState<JiaobeiRitual>`

#### 2.3 抽签卷轴展开（5s）
- [mobile/lib/core/animation/ritual/chouqian_ritual.dart]（新增）
  - 5 阶段：签筒摇晃 ±15° sin 波 → 签条跳出（弧形轨迹） → 卷轴展开（width 动画） → 签诗逐行毛笔书写（clipRect 左→右） → 金光描边
  - 卷轴纸面用米黄 `Color(0xFFE8D9B8)`，避免光污染
  - `_ChouqianRitualState extends RitualAnimationState<ChouqianRitual>`

#### 2.4 测字字形浮现 + 五行色染（5s）
- [mobile/lib/core/animation/ritual/cezi_ritual.dart]（新增）
  - 4 阶段：虚空粒子背景 → 「道」字浮现（easeOutBack 回弹） → 木行色（#6BAB6B）从字中心向外扩散染色 → 金色描边圆环描绘
  - 染色机制：`TextStyle(foreground: Paint()..shader = RadialGradient)`，精确控制 dye front
  - 演示字固定为「道」，与项目「志极 Jeenith」志/道主题呼应
  - `_CeziRitualState extends RitualAnimationState<CeziRitual>`

#### 2.5 仪式动画时长常量
- [mobile/lib/core/theme/animations.dart]：新增 4 个常量
  - `ritualMeihua = 4000ms` / `ritualJiaobei = 3000ms` / `ritualCezi = 5000ms` / `ritualChouqian = 5000ms`

### Phase 4.3：路由转场差异化

- [mobile/lib/core/animation/transitions/tech_transitions.dart]（新增）
  - `TechTransition.build(key, child, techId, transitionsEnabled)` 工厂方法
  - 10 术专属转场（switch by id）：
    - xiaoliuren：太极旋入（scale 0.5 + rotate 90°）
    - zhouyi：爻堆叠（slide up + scale Y）
    - meihua：数字合体（slide from left + easeOutBack）
    - jiaobei：抛物线（slide from top）
    - ziwei：辐射（scale 0.5 + rotate -30°）
    - qimen：九宫分块（scale 0.8 + slight rotation）
    - chouqian：卷轴展开（slide up + easeInOutCubic）
    - cezi：字符浮现（scale 0.6 + easeOutBack）
    - daliuren：双盘旋转（scale 0.7 + rotate 60°）
    - luopan：指针扫描（rotate -45° + scale 0.85）
  - `_custom()` 通用 builder 组合 fade + scale + rotate + slide
  - `_fade()` 兜底：transitionsEnabled=false 或未知 tech 时使用纯 FadeTransition

- [mobile/lib/router/app_router.dart]：接入差异化转场
  - `/tech/:id` 的 pageBuilder 调用 `TechTransition.build()`
  - 读 `configProvider.transitionsEnabled` 决定是否启用
  - 新增 import `tech_transitions.dart` 和 `config_providers.dart`

### Phase 5.1：DivinationLoadingIndicator

- [mobile/lib/shared/widgets/divination_loading_indicator.dart]（新增）

- [mobile/lib/features/settings/settings_page.dart]：替换为 `DivinationLoadingIndicator(size: 48)`

## 影响分析

- **仪式动画全覆盖**：10 术每术都有 3-6s 仪式入场动画（小六壬太极生六宫 + 5 套 v2.1.0 + 4 套 v2.2.0）
- **路由转场差异化**：10 术每术有专属视觉转场，关闭 `transitionsEnabled` 后退回纯 FadeTransition
- **加载指示器**：所有 `CircularProgressIndicator` 替换为中国风 `DivinationLoadingIndicator`
- **可控性**：4 个 sub-switch 独立可控（仪式/绘制/揭示/转场）+ 微交互动效总开关
- **性能**：所有新增组件用 RepaintBoundary 隔离；CustomPainter `shouldRepaint` 精确控制；AnimationController 严格 dispose
