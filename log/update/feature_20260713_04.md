# v1.5.0 — 主题切换 + 结果分享 + 历史导出

**类型**：feature
**日期**：2026-07-13 19:42
**影响范围**：主题系统、6 术页面 UI、历史页、设置页、共享组件、构建归档

## 变更描述

为志极补充三大完善性功能：深浅主题切换（system/light/dark 三态持久化）、6 术结果截图分享（RepaintBoundary → PNG → Share.shareXFiles）、历史记录三格式导出（JSON/Markdown/CSV）。这是多阶段实施计划的 Stage 4，聚焦用户体验与实用工具。

## 修改详情

### 主题切换链路

- `mobile/lib/core/config/app_config.dart`：
  - 追加 `import 'package:flutter/material.dart';`
  - 新增 `final ThemeMode themeMode;` 字段（默认 `ThemeMode.system`）
  - copyWith 追加 `ThemeMode? themeMode` 参数
- `mobile/lib/core/config/config_providers.dart`：
  - `build()` 读取 `prefs.getString('themeMode')` 并通过 `_parseThemeMode` 映射为 ThemeMode
  - 新增 `setThemeMode(ThemeMode v)` 方法：写入字符串 `'system'/'light'/'dark'` 到 SharedPreferences
- `mobile/lib/app.dart`：
  - 顶部读取 `ref.watch(configProvider).valueOrNull`
  - 新增 `_effectiveLight(context, config)` 方法：根据 themeMode + 系统亮度计算实际明暗
  - `MaterialApp.router` 的 `theme` 使用 `appTheme(isLight: isLightOuter)`
  - `builder` 内重新计算 `effectiveLight`（确保 MediaQuery context 正确）并传递给 ColoredBox + Starfield
- `mobile/lib/shared/widgets/starfield.dart`：
  - Starfield 新增 `final bool isLight;` 参数（默认 false）
  - `_StarPainter` 新增 `isLight` 字段
  - `paint()` 中 baseColor 根据 isLight 切换：浅色用 `Color(0xFF9B7A2A)`，深色用 `Color(0xFFD4A857)`
- `mobile/lib/core/theme/app_theme.dart`：上一会话已完成重写（AppColorsLight + AppFonts + appTheme({bool isLight})）

### 设置页 UI

- `mobile/lib/features/settings/settings_page.dart`：
  - 在「动画设置」分区前追加「◆ 外观」分区
  - `SegmentedButton<ThemeMode>` 三段（跟随系统/浅色/深色），每段带 Icon + Text
  - `onSelectionChanged` 调用 `ref.read(configProvider.notifier).setThemeMode(modes.first)`

### 结果分享按钮

- `mobile/lib/shared/widgets/share_result_button.dart`：新建文件
  - 接收 `GlobalKey boundaryKey` + 可选 `fallbackText`
  - `_share()` 通过 `findRenderObject()` 获取 `RenderRepaintBoundary`
  - `toImage(pixelRatio: 2.0)` → `toByteData(format: ui.ImageByteFormat.png)` → 写入临时文件
  - `Share.shareXFiles([XFile(file.path)])` 调用系统分享
  - 截图失败回退到 `Share.share(fallbackText)` 纯文本分享
  - 基于 `DarkButton` 保持视觉一致

### 6 术页面集成分享

- `mobile/lib/features/jiaobei/ui/jiaobei_page.dart`：import + `_boundaryKey` + Row 双按钮 + RepaintBoundary 包裹结果
- `mobile/lib/features/meihua/ui/meihua_page.dart`：同 jiaobei 模式
- `mobile/lib/features/ziwei/ui/ziwei_page.dart`：同 jiaobei 模式

### 历史导出

- `mobile/lib/core/history/history_export.dart`：新建文件
  - `exportJson()`：完整保真 JSON
  - `exportMarkdown()`：表格 + 详情块
  - `exportCsv()`：UTF-8 BOM 确保 Excel 正确识别

## 影响分析

- 主题切换让用户在深色/浅色环境自由选择，浅色主题保持金色卜算美学
- 结果分享让用户可以一键将卜算结果截图分享给朋友，提升传播性
- 历史导出支持三种格式：JSON 用于备份/导入、Markdown 用于阅读、CSV 用于 Excel 分析
- Starfield 适配浅色主题避免星点不可见
- 桌面端 sliver 页面（周易/小六壬）牺牲 lazy 渲染换取截图完整性，对当前内容量影响可忽略
- 思源宋体因字体文件缺失跳过打包，主题切换功能完整可用
- flutter analyze 0 issue，APK + Windows zip 构建归档正常
