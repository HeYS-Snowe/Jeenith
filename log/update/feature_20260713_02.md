# v1.3.0 紫微斗数 v2 全套星曜安星

**类型**：feature
**日期**：2026-07-13 17:06
**影响范围**：mobile/lib/features/ziwei/、builds/、pubspec.yaml

## 变更描述

为紫微斗数实现完整的星曜排盘（14 主星 + 六吉六煞 + 博士十二神 + 神煞），在结果页展示环形命盘图与宫位详情。这是多阶段实施计划的第 2 阶段（共 6 阶段）。

## 修改详情

- **新增** `mobile/lib/features/ziwei/algorithm/star_placement.dart`：安星算法核心
  - `StarPlacement` 数据结构（name + category）
  - `StarChart` 数据结构（12 宫星曜列表，含 == 与 hashCode）
  - `placeStars()` 函数：紫微星定位、天府星对称、紫微系/天府系 14 主星、六吉六煞、博士十二神、神煞
- **新增** `mobile/lib/features/ziwei/ui/star_chart_painter.dart`：CustomPainter 环形命盘
  - 12 宫扇形（子位于底部，逆时针布列）
  - 命宫金色高亮、身宫蓝色高亮
  - 每宫显示地支+宫名+主星+辅星
  - 中央太极点+命宫干支+五行局
  - TextPainter 显式 dispose（项目硬约束）
- **修改** `mobile/lib/features/ziwei/algorithm/divine.dart`：
  - ZiweiResult 追加 `final StarChart stars` 字段
  - divine() 内计算 yearZhi/day/ju 并调用 placeStars()
  - 新增 `_juNum` 映射（金4/木3/水2/火6/土5）
- **修改** `mobile/lib/features/ziwei/ui/ziwei_page.dart`：
  - 删除 v1 占位文字
  - 替换 4×4 GridView 为 CustomPaint + StarChartPainter
  - 追加 _buildGongDetailRow() 宫位详情列表
  - 更新 _buildCopyText() 包含各宫星曜
  - 删除废弃的 _palaceCell() 方法
- **修改** `mobile/lib/features/ziwei/data/stars.dart`：
  - 修复 const Map 不能用 for 元素的问题（改为 final）
  - 添加 library; 声明修复 dangling doc comment
- **修改** `mobile/pubspec.yaml`：版本 1.2.0+9 → 1.3.0+10
- **新增** `builds/release_notes/release_notes_v1.3.0.md`
- **修改** `builds/build_history.json`：追加 v1.3.0 APK + Windows zip 记录
- **修改** `builds/release_history.json`：v1.2.0 isLatest=false，追加 v1.3.0 记录

## 影响分析

- 紫微斗数功能从 v1 基础排盘升级为 v2 全套星曜排盘
- 安星算法是确定性函数（无随机），不影响 TrueRandom 引擎约束
- StarChartPainter 遵守 CustomPainter TextPainter dispose 硬约束
- HistoryStore 写操作模式不变（仅 detail 文本扩展）
- 其他 5 个卜算术不受影响
