# v1.6.0 — 抽签求签 + 测字

**类型**：feature
**日期**：2026-07-13 20:42
**影响范围**：新功能模块、注册表、构建归档

## 变更描述

为志极新增两个卜算术：抽签求签（chouqian）与测字（cezi）。通过 DivinationTech 注册模式扩展，无需修改 core/ 或 shared/ 任何代码，仅在 registry 追加两行。这是多阶段实施计划的 Stage 5，聚焦扩展性。

## 修改详情

### 抽签求签（chouqian）

- `mobile/lib/features/chouqian/algorithm/divine.dart`：新建
  - `enum StickGrade`：5 等级枚举（上上/上/中/下/下下）+ label 扩展 + luckScore
  - `class ChouqianStick`：number/title/grade/poem/interpretation/detail 数据类
  - `class ChouqianResult`：stick + time
  - `const List<ChouqianStick> sticks`：30 签数据，覆盖五等级（上上 5/上 5/中 10/下 5/下下 5）
  - `const int stickCount = 30`
  - `ChouqianResult divine(int stickNumber, {DateTime? time})`：根据签号取签，越界回绕
- `mobile/lib/features/chouqian/ui/chouqian_page.dart`：新建
  - `ConsumerStatefulWidget`（访问 trueRandomProvider）
  - 竹签筒可视化：`_SticksPainter extends CustomPainter` 绘制筒内一束竹签 + 升起的签
  - 摇签动画：`AnimationController _shake`（1100ms）+ 阻尼振荡 `sin(8π·t) × (1-t) × 0.18`
  - 落签动画：`AnimationController _fall`（600ms）控制签升起 60 像素
  - `_onDraw()`：调用 `ref.read(trueRandomProvider).generate(count: 1, vmax: stickCount)` 取签号
  - 结果展示：签号 + 等级徽章（颜色分级）+ 签题 + 签诗（楷体竖排居中）+ 解曰 + 详注
  - CopyResultButton + ShareResultButton Row 布局 + RepaintBoundary 包裹结果
  - `_SticksPainter.paint()` 中 TextPainter 显式 dispose()（遵循硬约束）
- `mobile/lib/features/chouqian/chouqian_tech.dart`：新建
  - `class ChouqianTech extends DivinationTech`
  - `id='chouqian'`, `meta.sortOrder=6`, `meta.accentColor=AppColors.wood`（木色）

### 测字（cezi）

- `mobile/lib/features/cezi/algorithm/divine.dart`：新建
  - `enum WuxingAttr`：5 五行枚举（金/木/水/火/土）+ label/nature 扩展
  - `class CeziResult`：inputChar/strokes/wuxing/strokeAnalysis/poem/interpretation/detail/time
  - `const Map<String, int> strokeTable`：200+ 常用汉字笔画表（简体计画）
  - `int strokesOf(String char)`：取笔画，未收录按 unicode 哈希估算
  - `WuxingAttr wuxingByStrokes(int strokes)`：1/2 木，3/4 火，5/6 土，7/8 金，9/0 水
  - `CeziResult divine(String inputChar, {DateTime? time})`：纯函数式，输入确定即输出确定
  - 私有辅助：`_strokeAnalysis`（按笔画数判断结构特征）、`_poemFor`（按五行生成断语诗）、`_interpretation`、`_detailFor`
- `mobile/lib/features/cezi/ui/cezi_page.dart`：新建
  - `StatefulWidget`（无需 ref，纯函数式算法）
  - `TextEditingController _ctrl`（maxLength:1，居中大字输入）
  - 输入校验：`RegExp(r'^[\u4e00-\u9fa5]$')` 限制单个汉字，非汉字 SnackBar 提示
  - 大字展示：110×110 卡片 + 五行色描边
  - 三属性徽章：笔画 / 五行 / 性情（颜色：金/木/水/火/土 五行色）
  - 拆解 / 断语诗 / 解字 / 详注四段式展示
  - 测字要诀说明面板（凝神→浮现→拆字三步）
  - CopyResultButton + ShareResultButton Row + RepaintBoundary
- `mobile/lib/features/cezi/cezi_tech.dart`：新建
  - `class CeziTech extends DivinationTech`
  - `id='cezi'`, `meta.sortOrder=7`, `meta.accentColor=AppColors.waterDeep`（水深色）

## 影响分析

- 首页 8 张卡片（小六壬/周易/梅花/掷筊/紫微/奇门/抽签/测字），无 grid 布局问题（4 行 2 列）
- 抽签遵循 TrueRandom 引擎硬约束（多源 SHA256 混合熵）
- 测字纯函数式算法，无需 RNG，直接根据字符确定输出
- _SticksPainter 中 TextPainter 显式 dispose()，遵循项目硬约束
- 测字输入校验防注入：regex 限制单个 CJK 汉字
- 笔画表覆盖 200+ 常用字，未收录字按 unicode 哈希估算保证可重现
- flutter analyze 0 issue，APK + Windows zip 构建归档正常
- 注册模式验证：加新术 = 新建 feature 目录 + 实现 DivinationTech + registry 一行，无需改 core/shared
