# v1.2.0 卦辞爻辞数据与文本展示

**类型**：feature
**日期**：2026-07-13 13:48
**影响范围**：mobile/assets/yijing/, mobile/lib/data/yijing/, mobile/lib/features/zhouyi/, mobile/lib/features/meihua/, mobile/lib/main.dart, mobile/pubspec.yaml, builds/

## 变更描述

为周易 64 卦 + 384 爻建立完整的古文与白话注解数据资产，在周易（金钱卦）与梅花易数结果页展示卦辞与变爻爻辞。这是志极多阶段实施计划的第 1 阶段（共 6 阶段），聚焦内容深化。

## 修改详情

- [新建] mobile/assets/yijing/hexagrams.json：64 卦完整数据（卦辞原文 + 白话注解 + 爻辞数组），乾坤各 7 爻含用九用六，共 64 卦辞 + 386 爻辞
- [新建] mobile/lib/data/yijing/hexagram_texts.dart：HexagramTexts 单例加载器，rootBundle 预加载到内存 Map，提供 guaCi/guaCiNote/yaoCi/yaoCiNote/posName/yongCi 查询
- [修改] mobile/lib/main.dart：启动时调用 HexagramTexts.load() 预加载
- [修改] mobile/lib/features/zhouyi/ui/zhouyi_page.dart：_resultItems 追加本卦卦辞/变爻爻辞/变卦卦辞卡片；新增 _buildGuaCiCard/_buildYaoCiCard helper；_buildCopyText() 追加卦辞爻辞段落
- [修改] mobile/lib/features/meihua/algorithm/divine.dart：MeihuaResult 加 dongPos/dongYang getter（从经卦二进制值提取动爻阴阳）
- [修改] mobile/lib/features/meihua/ui/meihua_page.dart：_buildResult 追加本卦卦辞/动爻爻辞/变卦卦辞卡片；_buildCopyText() 追加卦辞爻辞段落
- [修改] mobile/pubspec.yaml：版本号 1.1.3+7 → 1.2.0+9，assets 追加 - assets/yijing/
- [更新] builds/build_history.json：追加 v1.2.0 APK + Windows zip 记录
- [更新] builds/release_history.json：v1.1.3 isLatest 改为 false，追加 v1.2.0 发布记录
- [新建] builds/release_notes/release_notes_v1.2.0.md

## 影响分析

- 周易/梅花结果页新增卦辞爻辞文本展示，不影响现有起卦算法
- HexagramTexts 在启动时预加载，不影响 UI 性能

- 复制结果文本变长，包含完整的卦辞爻辞原文与注解
