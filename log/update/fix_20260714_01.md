# v2.2.0+19 — 动效组件 Bug 修复（particle_system / ink_spread 编译错误）

**类型**：fix
**日期**：2026-07-14 22:30
**版本**：2.2.0+19
**影响范围**：core/animation/particles/particle_system.dart、core/animation/reveal/ink_spread.dart

## 变更描述

修复 v2.2.0 引入的两个动效组件编译错误：
1. `particle_system.dart` 中 `LightTrail` 使用 `LinearGradient.begin/end` 时传入了 `Offset` 类型，应为 `AlignmentGeometry`；`Path.getBoundsFromPath(path)` 方法不存在，应为 `path.getBounds()`。
2. `ink_spread.dart` 中存在未使用的 `import '../../theme/app_theme.dart'`。

## 修改详情

### [mobile/lib/core/animation/particles/particle_system.dart]

修复前（编译错误）：
```dart
final gradient = LinearGradient(
  begin: start,        // Offset —— 类型错误
  end: current,        // Offset —— 类型错误
  ...
);
...
..shader = gradient.createShader(Path.getBoundsFromPath(path))  // 方法不存在
```

修复后：
```dart
final path = Path()..moveTo(start.dx, start.dy)..lineTo(current.dx, current.dy);
final rect = path.getBounds();
// Guard against zero-size rect (vertical/horizontal line).
final w = rect.width == 0 ? 1.0 : rect.width;
final h = rect.height == 0 ? 1.0 : rect.height;
final gradient = LinearGradient(
  begin: Alignment(
    ((start.dx - rect.left) / w) * 2 - 1,
    ((start.dy - rect.top) / h) * 2 - 1,
  ),
  end: Alignment(
    ((current.dx - rect.left) / w) * 2 - 1,
    ((current.dy - rect.top) / h) * 2 - 1,
  ),
  ...
);
...
..shader = gradient.createShader(rect)
```

**修复要点**：
- 把 `Offset` 转换为 `Alignment` 相对坐标：`Alignment(x, y)` 范围 [-1, 1]，对应 [0, 1] 的 Rect
- 用 `path.getBounds()` 替代不存在的 `Path.getBoundsFromPath(path)`
- 增加零尺寸 rect 边界保护（垂直/水平线时 width 或 height 为 0）

### [mobile/lib/core/animation/reveal/ink_spread.dart]

- 删除未使用的 `import '../../theme/app_theme.dart'`

## 影响分析

- `LightTrail` 组件被紫微仪式动画（`ziwei_ritual.dart`）中 14 主星降落光尾使用
- 此前编译错误会让整个 v2.2.0+18 无法构建
