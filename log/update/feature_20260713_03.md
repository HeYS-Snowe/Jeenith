# v1.4.0 — 奇门遁甲 v2 四盘九宫

**类型**：feature
**日期**：2026-07-13 19:05
**影响范围**：奇门遁甲算法与 UI、构建归档、发布记录

## 变更描述

将奇门遁甲从 v1 基础排盘（阴阳遁+局数+节气+八字）升级为完整四盘排盘：天盘九星 + 地盘九干 + 人盘八门 + 神盘八神，含值符值使定位、九宫布局、中宫寄二宫规则。这是多阶段实施计划的 Stage 3。

## 修改详情

- `mobile/lib/features/qimen/algorithm/divine.dart`：
  - 新增 `QimenPlate` 类（dunType/ju/jieqi/bazi/lunarDisplay + 四盘数组 + 值符值使字段）
  - 新增常量表：palaceNames/starByPalace/menByPalace/shenOrder/ganChars/zhiChars
  - 新增工具函数：_ganzhiIndex（60甲子索引）、_nextPalace/_prevPalace（顺逆飞）、_palaceDistance（路径步数）
  - 重写 `divine()`：增加时辰干支解析、旬首定位、值符值使计算、四盘构建
  - 新增 `_buildPlate()` 私有函数：实现六仪三奇布盘、天盘转动、人盘转动、神盘布神
- `mobile/lib/features/qimen/ui/qimen_page.dart`：
  - 新增 `_luoshuDisplay` 常量：洛书九宫 3×3 展示顺序
  - `_buildResult()` 替换 v1 占位 9 宫格为四盘叠加显示 + 值符值使高亮卡片
  - 新增 `_buildPalaceCell()`：9 宫单格四层叠加（天盘星/地盘干/人盘门/神盘神），含值符值使描边高亮
  - 新增 `_buildPanTable()`：单盘详表横向 9 宫
  - 重写 `_buildCopyText()`：追加九宫四盘完整列表 + 值符值使信息
  - 删除"v1 基础排盘"占位文字
- `mobile/pubspec.yaml`：版本 1.3.0+10 → 1.4.0+11
- `builds/build_history.json`：追加 v1.4.0 APK + Windows zip 两条记录
- `builds/release_history.json`：v1.3.0 isLatest=false，追加 v1.4.0 isLatest=true（unpublished）
- `builds/release_notes/release_notes_v1.4.0.md`：发布说明文档
