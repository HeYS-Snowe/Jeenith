# 新增使用手册页面 + 圆角修复 + 历史复制 + 默认深色

**类型**：feature
**日期**：2026-07-13 23:30
**影响范围**：features/manual/、features/home/、features/settings/、features/history/、core/config/、app.dart、shared/widgets/decorative_panel.dart、router/app_router.dart、assets/icons/

## 变更描述

为志极新增四组改动：使用手册页面（含 SVG 手册图标、入口、详细 Q&A 与历法体系说明）、修复设置/首页/历史列表中圆角卡片视觉与点击区域不一致的问题、为历史记录每条记录增加复制功能、将默认主题模式从「跟随系统」改为「深色模式」。

## 修改详情

### 新增：使用手册页面（Task 1）

- [mobile/assets/icons/manual.svg]：新增 menu_book 风格手册 SVG 图标（24×24 viewBox，单 path + currentColor 染色）
- [mobile/lib/features/manual/manual_page.dart]：使用手册页面，分 5 个章节：
  - 基础使用：4 条使用方法
  - 时辰边界 Q&A：4 组 Q&A，含用户实际案例（2006-12-03 8:40-9:00 出生辰时还是巳时、8:58 填 8 还是 9、整点归属、出生时间不确定怎么办）
  - 历法体系区分：农历体系 vs 节气干支历体系 + 普遍错觉的成因
  - 各术数历法归属：志极 10 个术数的历法归属表（紫微/小六壬/抽签→农历；奇门/大六壬/梅花/周易→节气；风水罗盘/测字/掷筊→不依赖历法）
  - 实操提醒：5 条易错点提示
- [mobile/lib/router/app_router.dart]：追加 `/manual` 路由
- [mobile/lib/features/home/home_page.dart]：首页右上角追加 SvgIcon('manual') IconButton，与设置按钮组合在 Row 内

### 修复：圆角点击区域问题（Task 2）

- [mobile/lib/features/settings/settings_page.dart]：
  - `_card()` Container 加 `clipBehavior: Clip.antiAlias`
  - ExpansionTile 的 `shape: const Border()` → `RoundedRectangleBorder(borderRadius: BorderRadius.circular(14))`
  - `collapsedShape` 同上
  - 3 个 SwitchListTile 加 `shape: RoundedRectangleBorder(borderRadius: BorderRadius.circular(14))`
- [mobile/lib/features/home/home_page.dart]：
  - `_guideCard()` Container 加 `clipBehavior: Clip.antiAlias`
  - ExpansionTile 的 shape 和 collapsedShape 改为 RoundedRectangle
- [mobile/lib/shared/widgets/decorative_panel.dart]：Container 加 `clipBehavior: Clip.antiAlias`
- [mobile/lib/features/history/history_page.dart]：ListTile 加 `shape: RoundedRectangleBorder(borderRadius: BorderRadius.circular(16))`

### 新增：历史记录复制功能（Task 3）

- [mobile/lib/features/history/history_page.dart]：
  - 新增 `_copyEntry(HistoryEntry e)` 方法：组装"【术数名】摘要/时间/详情/备注"格式，调用 Clipboard.setData，SnackBar 提示
  - ListTile trailing 改为 Row(IconButton(Icons.copy_all) + Icon(chevron_right))，IconButton 触发 _copyEntry，不影响 ListTile onTap 进入详情
  - _openDetail 对话框 actions 中追加"复制"按钮（在删除和保存备注之前）

### 配置变更：默认主题模式改为「深色」（Task 4）

- [mobile/lib/core/config/app_config.dart]：`themeMode` 默认值从 `ThemeMode.system` 改为 `ThemeMode.dark`，影响 AppConfig 构造默认值和 defaults 静态常量
  - 同时修复一处语法 bug：`defaults` 中 `xiaoliurenCinematic = true` 改为 `xiaoliurenCinematic: true`（const 表达式应使用命名参数语法 `:`）
- [mobile/lib/core/config/config_providers.dart]：`_parseThemeMode` 默认 fallback 改为 `ThemeMode.dark`，新增 `case 'system'` 显式分支
- [mobile/lib/app.dart]：`_effectiveLight` 的 fallback 从 `ThemeMode.system` 改为 `ThemeMode.dark`
- 影响范围：仅对全新安装的用户生效；已存在的用户偏好（持久化在 SharedPreferences）不受影响

## 影响分析

- **新增功能**：使用手册页面 + SVG 图标 + 历史记录复制按钮
- **修复**：圆角卡片视觉与点击区域一致性问题，水波纹不再溢出圆角外
- **配置**：默认主题改为深色，新用户安装后开箱即用深色主题
- **兼容性**：现有用户偏好不受影响；DecorativePanel 加 clip 后所有使用该组件的页面（首页卡片、各术数结果页等）点击水波纹会被圆角 clip，视觉更整洁
- **可维护性**：使用手册内容直接嵌入 ManualPage（无外部数据源），后续更新只需修改本文件
