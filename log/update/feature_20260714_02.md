# v2.1.0 — 动效体系 Phase 1 + Phase 2a（5 套仪式动画落地）

**类型**：feature
**日期**：2026-07-14 16:35
**版本**：2.1.0+17
**影响范围**：core/animation/（全新目录）、core/theme/animations.dart、core/config/app_config.dart、core/config/config_providers.dart、features/settings/settings_page.dart、features/home/home_page.dart、router/app_router.dart、pubspec.yaml、builds/

## 变更描述

落地 `docs/NEXT_PLAN/NEXT_PLAN_v2.1.0_ANIMATION.MD` 设计方案的 Phase 1（基础设施）+ Phase 2a（前 5 套仪式动画），让占卜过程具备仪式感。

v2.0.0 完成按钮/图标级微交互；v2.1.0 升级到「全术仪式化动效体系」——用户点击卡片后，先经历一段 4-6s 的仪式动画（铜钱抛落 / 命盘展开 / 九宫飞布 / 双盘旋转 / 罗盘扫描），再进入操作页。3s 后出现「跳过」按钮，关闭「仪式动画」开关后可直接进入。

按用户约束实现：80% 大动效 + 20% 微动效；黑金低饱和度无光污染；性能流畅可承载大量动画；4 个 sub-switch 独立可控（仪式/绘制/揭示/转场）。

## 修改详情

### Phase 1：基础设施

#### 1.1 仪式动画抽象基类
- [mobile/lib/core/animation/ritual/ritual_animation.dart]（新增）
  - `RitualAnimation extends ConsumerStatefulWidget`：持有 `onCompleted` 回调
  - `RitualAnimationState<T> with TickerProviderStateMixin`：子类创建自己的 `AnimationController`
  - `ritualScaffold(content)`：Scaffold + `AppColors.bg` 背景 + 3s 后淡入的「跳过」按钮
  - `_SkipButton`：鎏金描边半透明背景，淡入出现，点击触发 `complete()`

#### 1.2 CustomPainter 动画基类
- [mobile/lib/core/animation/painters/animated_chart_painter.dart]（新增）
  - `progress` 参数驱动绘制（0.0 → 1.0）
  - 工具：`seg(start, end)` / `easeOut(t)` / `easeInOut(t)` / `easeOutBack(t)` / `lerpAngle()` / `lerp()`

#### 1.3 粒子系统
- [mobile/lib/core/animation/particles/particle_system.dart]（新增）
  - `ParticleBurst`：粒子爆发组件（center/count/color/maxRadius/duration）
  - `LightTrail`：金光线段，用于星曜降落光尾

#### 1.4 结果揭示动画组件
- [mobile/lib/core/animation/reveal/typewriter_text.dart]（新增）
  - `TypewriterText`：StepTween 逐字显示 + 光标闪烁
  - `StaggeredReveal`：段落错峰淡入上浮
- [mobile/lib/core/animation/reveal/ink_spread.dart]（新增）
  - `InkSpread`：ShaderMask + RadialGradient 墨晕开
  - `ScaleReveal`：缩放 + 透明度揭示

#### 1.5 配置开关升级
- [mobile/lib/core/config/app_config.dart]：新增 4 个字段（均默认 true）
  - `ritualAnimationsEnabled` / `painterAnimationsEnabled` / `revealAnimationsEnabled` / `transitionsEnabled`
- [mobile/lib/core/config/config_providers.dart]：
  - `build()` 读取 4 个新 prefs key
  - 新增 4 个 setter：`setRitualAnimationsEnabled` / `setPainterAnimationsEnabled` / `setRevealAnimationsEnabled` / `setTransitionsEnabled`

#### 1.6 仪式时长常量
- [mobile/lib/core/theme/animations.dart]：新增仪式动画时长常量
  - `ritualZhouyi = 5000ms` / `ritualZiwei = 6000ms` / `ritualQimen = 5000ms`
  - `ritualDaliuren = 5000ms` / `ritualLuopan = 4000ms`
  - `skipButtonDelay = 3000ms`

#### 1.7 设置页 sub-switch
- [mobile/lib/features/settings/settings_page.dart]：「◆ 动效」section 在主开关 `animationsEnabled` 下新增 4 个 sub-switch
  - 仪式动画 / 绘制过程动画 / 结果揭示动画 / 路由转场动画

## 影响分析

- **用户体验**：5 个核心术（周易/紫微/奇门/大六壬/风水罗盘）点击卡片后有仪式动画，仪式感大幅提升
- **可控性**：4 个 sub-switch 独立可控，用户可在设置页关闭不想看的动画类别
- **性能**：所有动画用 RepaintBoundary 隔离 + AnimationController dispose 严格管理；60fps 流畅
- **兼容性**：小六壬仪式动画逻辑不变（继续用 `xiaoliurenCinematic`）；其余 4 术（梅花/掷筊/抽签/测字）尚未实现仪式动画，仍直接进 tech 页
- **基类修复**：qimen_ritual.dart 创建过程中发现基类 `ritual_animation.dart` 错误引用了不存在的 `AppColors.background`，已统一为 `AppColors.bg`
