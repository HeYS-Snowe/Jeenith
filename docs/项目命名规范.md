# 校园危险行为预警系统 - 项目命名规范

> 本文件记录 APP 的统一命名，供工程开发、文档、商店上架使用。

## 组织信息 Organization

| 项 Item | 内容 Content |
|---|---|
| 组织名称 Organization | Qore（叩心） |
| 名称来源 Etymology | Qore 取自 "QoCore"，"Qo" 为"叩"的音译 |
| 口号/寓意 Slogan | 叩问本心，不忘初心 |
| 包名前缀 Package Prefix | com.qore |

## 命名总表

| 项 | 内容 | 说明 |
|---|---|---|
| 中文名 | 校卫哨兵 | APP 显示名 |
| 英文名 | Campus Sentinel | 国际化 / 商店副标题 |
| 简称 | 哨兵 / Sentinel | 内部代号、文档缩写 |
| 应用包名 | com.qore.campussentinel | 组织前缀 com.qore |

## 名字寓意

- 校卫：校园保卫，贴合校园安保编制
- 哨兵：24h 站岗放哨、主动预警，突出"监测 - 预警"功能定位

## Logo 概念

- 元素：盾牌 + 瞭望塔/哨岗 + 雷达扫描线
- 盾牌：守护属性
- 雷达扫描线：AI 识别 / 主动扫描

## 配色方案

| 用途 | 颜色 | 说明 |
|---|---|---|
| 主色 | 深蓝 | 安全 / 科技感 |
| 强调色 | 警示橙 | 预警提示 |
| 警报状态 | 红色 | 高危预警 |
