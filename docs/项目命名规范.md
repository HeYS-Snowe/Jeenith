## 名字寓意

- 志：志于本心，叩问本心
- 极：知于极处，不忘初心
- Jeenith：志极的音译，兼有 Zenith（顶点、极处）之意

## 品牌精神

志于本心，知于极处 —— *Question the core. Return to origins.*

## Logo 概念

- 元素：太极 + 星象 + 卜算符号
- 深色背景 + 鎏金描边，体现东方玄学与质感
