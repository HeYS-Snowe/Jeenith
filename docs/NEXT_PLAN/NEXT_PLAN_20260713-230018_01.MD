## 📋 模块一：核心微交互升级（UI / UX 深化）

【主要对应：任务1-1、1-2、1-3，以及用户指定的图标与按钮重构】

- [x] **按钮物理反馈与样式重写**：
  - [x] 移除原生 `ElevatedButton` 的默认水波纹。改用 `AnimatedContainer` + `GestureDetector`，统一封装 `GoldButton` 和 `DarkButton`。
  - [x] **按动瞬间**：缩小至 0.95 倍，阴影偏移量降低。
  - [x] **抬起瞬间**：使用 `Curves.easeOutBack` 弹回 1.0 倍，阴影回归。过渡时长为 200ms - 300ms。
  - [x] *注意*：颜色变化必须遵循 `oklch` 色彩体系（黑金低饱和度）。
- [x] **图标状态切换动画（+ ↔ x）**：
  - [x] 为列表折叠/展开、功能性动作实现该动画。
  - [x] 底层：组合 `AnimatedRotation` + `FadeTransition`。旋转角度为 0° → 90°（或 180°）。
  - [x] 加成：点击瞬间带轻微 `AnimatedScale` 呼吸反馈。
- [x] **其他微交互补充**：
  - [x] 首页「使用说明」卡片展开收起的缓动曲线修正。
  - [x] 卜算结果页卡片出现时的错峰上浮动画（避免"平铺直叙"的加载感）。
- [x] **设置页动效配置集成**：
  - [x] 所有上述新增的动效，统一封装进 `core/theme/animations.dart` 中。
  - [x] 在设置页为它们（或整体）添加"动效开启/关闭"的开关配置，并持久化到 `SharedPreferences`。
