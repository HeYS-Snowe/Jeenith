## 1. 设计概述 Design Overview

### 1.1 设计目标 Design Goals

- **仪式感**：起卦、摇爻、掐指、布盘过程具有仪式感动效
- **精致**：现代、精致的视觉呈现，不古板
- **统一**：12 术统一交互范式，降低学习成本
- **可控**：动效可全局开关 + 4 AnimationKind 细分控制
- **跨端**：Android + Windows 桌面体验一致

### 1.2 设计原则 Design Principles

| 原则 Principle | 描述 Description |
|-------------|---------------|
| 仪式而非表单 | 起卦过程是叩问本心的仪式 |
| 精致而非古板 | 现代审美 + Flutter 跨端 |
| 统一而非割裂 | 12 术统一交互范式 |
| 可控而非强制 | 动效可关闭，主题可切换 |
| 隐私而非侵入 | 无账号、本地存储 |

### 1.3 品牌精神 Brand Spirit

**志于本心，知于极处 —— Question the core. Return to origins.**

- **志于本心**：每一次起卦、摇爻、掐指、布盘都是叩问本心的仪式
- **知于极处**：算法规范、真随机引擎、精致体验，知其极致

## 2. 页面结构 Page Structure

### 2.1 整体页面树 Page Tree

```
JeenithApp
├── 首页 Home（Starfield + 选术 grid）
│   ├── 小六壬 → 操作页 → 仪式页 → 结果页
│   ├── 周易 → 操作页 → 仪式页 → 结果页

│   ├── 紫微斗数 → 输入页 → 排盘页
│   ├── 奇门遁甲 → 输入页 → 排盘页
│   ├── 抽签 → 操作页 → 仪式页 → 结果页
│   ├── 测字 → 输入页 → 结果页
```

## 11. 交互流程 Interaction Flow

### 11.1 周易起卦完整流程 Zhouyi Flow

```
1. 首页
   ↓ 点击"周易"
2. 周易操作页
   ↓ 点击"摇卦"
3. 周易仪式页（Phase 3 摇卦动画）
   ├─ 第 1 爻（真随机 + 动画）
   ├─ 第 2 爻
   ├─ ...
   └─ 第 6 爻
   ↓ 6 爻完成
4. 周易结果页（Phase 4 卦象揭示）
   ├─ 本卦 / 变卦 / 互卦
   ├─ 卦辞 / 爻辞
   ├─ 吉凶判词
   ├─ 历史自动写入
   └─ [复制] [分享] [返回]
```

### 11.2 紫微排盘完整流程 Ziwei Flow

```
1. 首页
   ↓ 点击"紫微斗数"
2. 紫微输入页
   ├─ 输入生辰（农历/公历）
   ├─ 选择性别
   ↓ 点击"排盘"
3. 紫微排盘页（Phase 3 命盘绘制）
   ├─ 12 宫绘制
   ├─ 14 主星安星
   ├─ 辅星安星
   ↓ 安星完成
4. 紫微结果页（Phase 4 命盘揭示）
   ├─ 命盘图
   ├─ 星曜解读
   ├─ 历史自动写入
   └─ [复制] [分享] [返回]
```

### 11.3 风水罗盘流程 Luopan Flow

```
1. 首页
   ↓ 点击"风水罗盘"
2. 罗盘页
   ├─ 检测传感器
   ├─ [可用] 申请权限 → 启动罗盘
   └─ [不可用] 降级提示
3. 罗盘实时显示
   ├─ 实时方位角
   ├─ 24 山指示
   └─ 方位解读
```

### 11.4 设置流程 Settings Flow

```
1. 首页
   ↓ 点击"设置"
2. 设置页
   ├─ 切换主题
   ├─ 切换动效全局开关
   └─ 切换 AnimationKind 细分
3. 配置持久化（shared_preferences）
4. 全局生效
```
