## 附录 Appendix

### 严重级别定义 Severity Definition

| 级别 Severity | 定义 Definition | 示例 Example |
|-------------|-------------|-------------|
| S-致命 Fatal | 系统崩溃、数据丢失、安全漏洞 | 无法登录、数据库损坏 |
| A-严重 Critical | 主要功能无法使用 | 支付失败、无法下单 |
| B-一般 Major | 功能可用但有缺陷 | 显示错误、校验问题 |
| C-轻微 Minor | 小问题、不影响使用 | 文案错误、样式问题 |
