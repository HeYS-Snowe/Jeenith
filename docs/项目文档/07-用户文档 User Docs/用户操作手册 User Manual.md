## 3. 功能操作指南 Feature Guide

### 3.1 用户账户 User Account

#### 3.1.1 修改个人信息

| 步骤 Step | 操作说明 Operation |
|----------|------------------|
| 1 | 点击右上角头像 → "个人中心" |
| 2 | 点击"编辑资料"按钮 |
| 3 | 修改需要更新的信息 |
| 4 | 点击"保存"按钮 |

#### 3.1.2 修改密码

| 步骤 Step | 操作说明 Operation |
|----------|------------------|
| 1 | 进入"个人中心" → "安全设置" |
| 2 | 点击"修改密码" |
| 3 | 输入旧密码和新密码 |
| 4 | 点击"确认修改" |

### 3.2 产品浏览 Product Browsing

#### 3.2.1 搜索产品

| 步骤 Step | 操作说明 Operation |
|----------|------------------|
| 1 | 在首页搜索框输入关键词 |
| 2 | 点击"搜索"按钮或按回车 |
| 3 | 浏览搜索结果 |

#### 3.2.2 查看产品详情

| 步骤 Step | 操作说明 Operation |
|----------|------------------|
| 1 | 在产品列表点击产品图片/名称 |
| 2 | 查看产品详细信息 |
| 3 | 选择规格（如需要） |
| 4 | 点击"加入购物车" |

### 3.3 订单操作 Order Operations

#### 3.3.1 下单流程

```
浏览商品 → 加入购物车 → 结算 → 确认订单 → 支付 → 完成下单
```

#### 3.3.2 查看订单

| 步骤 Step | 操作说明 Operation |
|----------|------------------|
| 1 | 进入"个人中心" → "我的订单" |
| 2 | 查看订单列表 |
| 3 | 点击订单查看详情 |

## 4. 常见问题 FAQ

### 4.1 账户相关 Account Related

**Q1: 忘记密码怎么办？**

A: 点击登录页面的"忘记密码"链接，按提示重置密码。

**Q2: 如何修改注册邮箱？**

A: 进入"个人中心"→"账户设置"，点击"修改邮箱"。

### 4.2 购物相关 Shopping Related

**Q1: 如何申请退款？**

A: 进入订单详情，点击"申请退款"，填写退款原因。

**Q2: 支付失败怎么办？**

A: 请检查支付余额，或联系客服。
