## 1. 应用概述 App Overview

### 1.1 应用简介 App Introduction

志极 Jeenith 是一款**叩问本心的卜算合集**移动应用，由 Qore Origins（叩心）出品。应用以「志于本心，知于极处」为品牌精神，口号「叩问本心，不忘初心」，将 12 种传统术数以现代移动体验重新呈现。

**核心特色**：

- **12 种传统术数**：小六壬、周易、梅花易数、掷筊、紫微斗数、奇门遁甲、抽签、测字、大六壬、风水罗盘、八字推演、测名字
- **真随机引擎**：三源熵混合（系统 + 触摸轨迹 + 在线真随机），保证起卦的随机性与仪式感
- **仪式感动效**：入场仪式、路由转场、绘制过程、结果揭示四阶段动效，可按需开关
- **隐私零外露**：纯客户端架构，无后端，所有数据本地存储
- **双端覆盖**：Android 移动端 + Windows 桌面端

### 1.2 适用平台 Supported Platforms

| 平台 Platform | 版本要求 Version | 说明 Description |
|-------------|---------------|---------------|
| Android | 5.0+（minSdk 21）| 移动端完整功能 |
| Windows | 10+ | 桌面端（风水罗盘除外，无磁力计硬件）|

### 1.3 应用入口 App Entry

| 平台 Platform | 入口 Entry |
|-------------|----------|
| Android | 安装 APK 后桌面图标「志极」|
| Windows | 解压 ZIP 后运行 `jeenith.exe` |

## 2. 安装与启动 Installation & Launch

### 2.1 Android 安装 Android Installation

```
步骤1：获取 APK 文件（从 GitHub Releases 下载）
步骤2：手机设置 → 安全 → 允许「未知来源」安装
步骤3：点击 APK 文件 → 安装
步骤4：安装完成 → 桌面出现「志极」图标
步骤5：点击图标启动应用
```

**首次启动**：

- 应用会弹出「使用方法」引导对话框（仅首次，关闭后不再弹出）
- 关闭引导后进入首页，可看到 12 种术数的选术卡片

### 2.2 Windows 安装 Windows Installation

```
步骤1：从 GitHub Releases 下载 Windows ZIP 包
步骤2：解压 ZIP 到任意目录（如 D:\Jeenith\）
步骤3：进入解压目录 → 运行 jeenith.exe
步骤4：桌面窗口自动居中，尺寸为屏幕 40%
```

**桌面端说明**：

- 窗口尺寸可调整，有最小尺寸约束
- 风水罗盘功能在桌面端不可用（无磁力计硬件）
- 触摸轨迹熵退化为鼠标轨迹（不影响起卦，仅仪式感略减）

### 2.3 卸载 Uninstallation

| 平台 Platform | 卸载方式 Method |
|-------------|---------------|
| Android | 设置 → 应用 → 志极 → 卸载 |
| Windows | 删除解压目录即可（绿色软件，无注册表残留）|

**数据清除**：卸载会清除所有本地数据（历史记录、设置）。如需保留，请先导出历史记录。

## 4. 使用方法引导 Usage Guide

### 4.1 引导卡片内容 Guide Content

首页的「使用方法」引导卡片包含 4 条核心建议：

**建议 1：单一卜算术起卦**
> 第一次问大致的问题，第二次问细致的问题。

**建议 2：多卜算术组合使用**
> 先用一种起卦问大致的问题，再用另外一种起卦问细节。

**建议 3：AI 辅助解读**
> 可以将卦象的完整截图和问题一并发送给 AI，也可以自己查资料。

**建议 4：起卦时机**
> 点击起卦按钮前，心里默念问题，集中注意力，注意力达到顶点的一刹那起卦。

### 4.2 首次引导 First-Time Guide

- 首次启动应用时，会自动弹出「使用方法」对话框
- 关闭对话框后，使用 SharedPreferences 记录「已读」状态，不再弹出
- 引导内容始终可在首页的引导卡片中查看（可展开/收起）

### 4.3 起卦心法 Divination Mindset

| 原则 Principle | 说明 Description |
|-------------|---------------|
| 专注 | 起卦前默念问题，集中注意力 |
| 顶点起卦 | 注意力达到顶点的一刹那点击起卦按钮 |
| 一次一问 | 同一术数不要反复起卦问同一问题 |
| 组合深入 | 用不同术数从不同角度问同一件事 |
| 戒贪戒惧 | 卦象为参考，决策在本心 |

## 10. 常见问题 FAQ

### 10.1 安装与启动 Installation & Launch

**Q1: Android 安装时提示「未知来源」怎么办？**

A: 手机设置 → 安全 → 允许「未知来源」安装应用。志极是开源软件，可放心安装。

**Q2: Windows 启动时窗口太大/太小怎么办？**

A: 窗口尺寸可手动拖动调整。最小尺寸有约束，避免过小影响操作。

**Q3: 首次启动的引导弹窗关掉了，怎么再看？**

A: 首页的「使用方法」引导卡片始终可见（可展开/收起）。或在设置中心「清除引导状态」后会重新弹出。

### 10.2 起卦相关 Divination

**Q1: 起卦结果不准怎么办？**

A: 志极采用三源真随机引擎（系统 + 触摸 + 在线），随机性有保障。若对结果有疑，建议：
- 检查问题是否明确（一次一问，避免模糊）
- 检查起卦时机（注意力顶点时起卦）
- 用不同术数组合验证（先大致后细节）

**Q2: 同一问题可以反复起卦吗？**

A: 不建议。传统原则「不疑不卜，一卜不疑」。若首次结果不清晰，可换一种术数从不同角度问。

**Q3: 风水罗盘指针不动/不准？**

A: 
- 确认在 Android 端使用（Windows 桌面端无磁力计）
- 远离金属/电子设备（磁场干扰）
- 水平持握设备
- 检查是否授权传感器权限

**Q4: 多种术数如何组合使用？**

A: 引导卡片建议：先用一种起卦问大致的问题，再用另外一种起卦问细节。例如：
- 先周易问「事业方向」→ 再小六壬问「下周面试结果」
- 先紫微看「命盘大势」→ 再八字看「流年运势」

### 10.3 功能功能 Features

**Q1: 历史记录会丢失吗？**

A: 不会。v2.3.1 已修复历史存储的原子性问题，快速连续卜算也不会丢数据。卸载应用会清除数据，请先导出。

**Q2: 动效卡顿怎么办？**

A: 设置中心 → 动效设置：
- 关闭「动效总开关」一键关闭所有动效
- 或关闭「结果揭示」「入场仪式」等耗性能的动效

**Q3: 如何将卦象发给 AI 解读？**

A: 在结果页点击「复制」按钮 → 粘贴到 AI 对话框 → 附上问题 → 发送。AI 会根据卦象文本辅助解读。

**Q4: 测字/测名字找不到某些字？**

A: 笔画数据库可能未覆盖所有生僻字。如遇缺失，欢迎在 GitHub Issues 反馈。

### 10.4 跨端问题 Cross-Platform

**Q1: Windows 桌面端为什么没有风水罗盘？**

A: 桌面电脑无磁力计硬件，sensors_plus 的磁力计 API 仅移动端可用。后续版本会探索鼠标拖拽模拟方位角输入。

**Q2: Windows 桌面端的随机性和手机端一样吗？**

A: 真随机引擎的三源中，系统熵（Random.secure）与在线熵（random.org）双端一致。触摸熵在桌面端退化为鼠标轨迹，随机性略低，但有系统熵 + 在线熵双保障，总体随机性有保证。
