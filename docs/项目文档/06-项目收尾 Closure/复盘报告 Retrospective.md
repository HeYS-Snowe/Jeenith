## 1. 复盘概述 Retrospective Overview

### 1.1 复盘目的 Retrospective Purpose

| 目的 Purpose | 说明 Description |
|-------------|---------------|
| 总结经验 | 总结项目中的成功经验和失败教训 |
| 持续改进 | 发现问题并提出改进措施 |
| 知识沉淀 | 将经验转化为团队知识资产 |

### 1.2 复盘方法 Retrospective Method

本次复盘采用 **4L方法**：

- **Liked** 做得好的方面
- **Learned** 学到的经验
- **Lacked** 遗憾和不足
- **Longed for** 期望改进的方面
