## 概述

v2.6.0 新增第 13 种卜算术——**称骨算命**（袁天罡称骨论命）。以农历生辰查年、月、日、时四骨重，相加得总骨重（单位精确到「钱」），对照称骨歌 52 档命格断一生荣枯。与既有八字、紫微互补，是民间流传最广的命理速断法。`flutter analyze` 0 issue。

## 一、称骨算命

### 算法
- **年骨重**：60 甲子年各一重量（甲子一两二钱 … 癸亥六钱）
- **月骨重**：农历正月至腊月（正六钱、三一两八钱 …）
- **日骨重**：农历初一到三十（初一五钱 … 三十六钱）
- **时骨重**：十二时辰（子时一两六钱 … 亥时六钱）
- 四项相加得总骨重（以整数「钱」存储，避免浮点误差），对照称骨歌 52 档（二两一钱 ~ 七月二钱）

### 内容
- **52 档命格**：每档含命格名 + 称骨歌（诗）+ 男命详解 + 女命详解
- 性别选择：按性别展示对应男命/女命断辞
- 四骨重明细：年（甲子 一两二钱）/ 月 / 日 / 时 逐项 + 总骨重
- 数据来源：袁天罡称骨论命（公知固定表，零流派分歧）

### 交互
- 公历年月日时输入 + 一键获取当前时间
- 历史记录预览：恢复生辰 + 性别 + 重算命格
