# v2.11.1 — 结果输出体验增强 + 历史记录搜索/筛选/批量删除

## 概述

本版本打磨「结果输出」链路：复制按钮文字过渡动画、复制成功提示的视觉升级、分享图的品牌署名水印；并为历史记录页增加搜索、按术数筛选与批量删除能力。

## 新增与改进

### 1. 复制按钮文字过渡动画

点击「复制结果」后，图标已有旋转缩放切换（copy_all → check），现补齐文字过渡：「复制结果」↔「已复制」以淡入 + 轻微上滑切换（300ms），与图标动画呼应。

为此 `DarkButton` 新增可选 `label` 参数（自定义文字 widget，自动套用按钮标签样式），不影响其他用法。

### 2. 复制成功 SnackBar 升级

由纯文字提示升级为 ✓ 图标 + 主题色卡片 + 圆角金边，反馈更精致。历史页复制记录的提示同步升级。

### 3. 分享图品牌署名水印

分享出去的 PNG 此前只有背景 + 金边、无文字。现于图底部居中加「志极 Jeenith · 叩问本心」小金字（思源宋体），每次分享都自带品牌传播。

### 4. 历史记录搜索 / 筛选 / 批量删除

- **搜索**：顶部搜索框，按摘要 / 术数名 / 详情 / 备注 实时过滤
- **按术数筛选**：横向 chip 一键筛某术历史（出现 2 种以上术数时显示）
- **批量删除**：长按列表项进入多选模式，可全选 / 取消全选 / 批量删除
