**版本**：2.5.0+29
**状态**：feature

## 概述

v2.5.0 是「断卦核心」补全版本，深化两大复杂术数——紫微斗数补齐四化/大限/长生/补煞，奇门遁甲补齐天盘干与格局断辞。此前紫微「只能看星不能断」（无四化）、奇门「只有盘没有判」（无格局），本版本补上断卦逻辑闭环。`flutter analyze` 0 issue。

## 一、紫微斗数深化（断卦核心）

### 1. 四化（化禄 / 化权 / 化科 / 化忌）
- 年干 → 四化 10 组标准表（甲廉破武阳、乙机梁紫阴 …）
- 星盘主星名旁标四色角标：禄金 · 权火 · 科水 · 忌灰
- 新增「生年四化」解读区：标注每化星曜落宫 + 含义（化禄=财运机会、化忌=阻碍是非等）
- 四化是紫微动态吉凶的核心，盘面由此「活」起来

### 2. 大限（十年大运）
- 五行局起运（水二 / 木三 / 金四 / 土五 / 火六岁）
- 阳男阴女顺行、阴男阳女逆行（输入性别决定方向）
- 每宫标注十年大限年龄段（如「限 24-33 岁」）
- 排盘区显示「大限顺行 / 逆行」

### 3. 长生十二神
- 命局五行起长生（水申 / 木亥 / 火寅 / 金巳 / 土申）
- 长生→沐浴→冠带→临官→帝旺→衰→病→死→墓→绝→胎→养，同大限方向布盘
- 每宫详情标注该宫长生神

### 4. 补充神煞（8 颗）
- 月系：天刑（酉起顺生月）、天姚（丑起顺生月）
- 年干系：天官、天福
- 年支系：天巫、孤辰、寡宿
- 空亡：截路空亡（年干对，双宫）

## 二、奇门遁甲深化（盘 → 断）

### 1. 补全天盘九干
- 地盘九干按天盘转动量（与九星同向同量）同步偏移
- 四盘详表新增「天盘九干」一行，干组合格局有据可循

### 2. 格局断辞（4 吉 + 7 凶）
- **吉格**：青龙返首（戊加值符）、飞鸟跌穴（乙加值符）、玉女守门（丁加值使）、三诈格（奇+吉门+吉神）
- **凶格**：青龙逃走（乙加辛）、白虎猖狂（辛加乙）、朱雀投江（丁加癸）、腾蛇妖娇（癸加丁）、太白入荧（庚加丙·贼来）、荧入太白（丙加庚·贼去）、三奇入墓、门迫（门克宫）
- 新增「格局断辞」区：吉/凶标签 + 落宫 + 文字断辞；无格时提示以星门生克论断
