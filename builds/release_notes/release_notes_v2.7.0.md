## 一、太乙神数

### 排盘算法（《太乙金镜式经》）
- **积年**：10153917 + 公元年（太极上元起算）
- **局数**：积年 %360 余数再 %72，得 1-72 局
- **阴阳遁**：冬至后阳遁、夏至后阴遁
- **太乙行宫**：三年一宫，24 年一周，不入中五
- **文昌（天目）**：积年 %18，从武德顺时针走（16 步周期，阴德/大武停一步）
- **计神**：年支六合（合神）顺时针下一间辰
- **始击（地目）**：文昌顺时针走「计神到艮宫」距离步数
- **主算/客算**：文昌/始击到太乙途经宫间数之和（间位 +1）
- **主客大将**：算数个位（0 则 %9）；**参将**：大将宫数 ×3 个位
- **格局**：迫 / 格 / 杜塞 / 囚 / 关 五类

### 验证
以 2024（阳遁 53 局）/ 2025 年盘逐项验证：主算 38 / 客算 25 / 太乙 2 宫 / 文昌坤大武 / 始击亥大义（2025 子地主）/ 计神戌（2025 酉）/ 主大将 8 宫 / 客大将中 5，**全部吻合**权威排盘。

### 界面
- **5×5 十六间辰盘**：外圈 16 间辰各配一神与宫数，太乙/文昌/始击/计神/主客将落位高亮标记
- **主客神将详表**：文昌/始击/计神 + 主客算/大将/参将
- **格局断辞区**：识别迫/格/杜塞/囚/关并给出含义
- 首次使用引导遮罩 + 历史预览接入
