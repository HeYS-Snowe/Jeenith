# v1.4.0 — 奇门遁甲 v2 四盘九宫

**版本**：1.4.0+11
**状态**：feature（多阶段计划第 3 阶段 · 内容深化）
**构建日期**：2026-07-13

## 概述

将奇门遁甲从 v1 的"阴阳遁+局数+节气+八字"基础信息升级为完整四盘（天/地/人/神）+ 值符值使 + 九宫布局。这是志极多阶段实施计划的第 3 阶段（共 6 阶段），聚焦内容深化。

## 新增功能

### 完整四盘排盘

- **天盘九星**：天蓬/天芮/天冲/天辅/天禽/天心/天柱/天任/天英，按洛书九宫本位定位，值符星随天盘转动
- **地盘九干**：六仪（戊己庚辛壬癸）+ 三奇（丁丙乙），戊起局数宫，阳遁顺布六仪逆布三奇，阴遁反之
- **人盘八门**：休/死/伤/杜/开/惊/生/景，值使门按时辰在旬中的序号转动
- **神盘八神**：直符/螣蛇/太阴/六合/白虎/玄武/九地/九天，直符起于值符天盘位置，阳遁顺布阴遁逆布

### 值符值使定位

- 根据时辰干支推算 60 甲子旬首
- 旬首遁干（甲遁在六仪下）所在宫为值符宫
- 值符星 = 值符宫的本位星
- 值使门 = 值符宫的本位门
- 值使转动后所在宫由时辰在旬中的序号决定

### 中宫寄二宫

- 当六仪/八门/八神落中宫（5宫）时，寄二宫（坤宫）显示
- 符合传统奇门遁甲规则

### UI 升级

- 9 宫格采用洛书布局：4巽 9离 2坤 / 3震 5中 7兑 / 8艮 1坎 6乾
- 每宫四层叠加显示：天盘星（金）+ 地盘干（白）+ 人盘门（绿）+ 神盘神（紫）
- 值符宫金色描边高亮 + "符"标识
- 值使宫绿色描边高亮 + "使"标识
- 9 宫格下方追加"四盘详表"：天/地/人/神 各一张表
- 复制结果文本包含九宫四盘完整列表 + 值符值使信息

## 技术要点

- 60 甲子索引算法：通过干支索引求解 `i % 10 == ganIdx, i % 12 == zhiIdx`
- 洛书顺飞路径：1→2→3→4→5→6→7→8→9→1
- 洛书逆飞路径：1→9→8→7→6→5→4→3→2→1
- 阳遁顺布六仪逆布三奇，阴遁反之
- 天盘转动：值符星移到时干所在宫，其他星同步偏移
- 人盘转动：值使门按 posInXun 步同步转动
- 神盘：直符起于值符天盘位置（即天盘转动后的值符宫）
- flutter analyze 0 issue
