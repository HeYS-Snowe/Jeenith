# v1.7.0 — 大六壬全套 + 风水罗盘

**版本**：1.7.0+14
**状态**：feature（多阶段计划第 6 阶段 · 终章）

## 概述

为志极新增两个卜算术：大六壬与风水罗盘。这是多阶段实施计划（v1.2.0 → v1.7.0）的收官阶段——大六壬完整实现了三传四课体系，风水罗盘首次引入磁力计硬件接入。

## 新增功能

### 大六壬（daliuren · sortOrder 8）

- **四课推演**：日干寄宫（甲寅乙辰丙戊巳丁己未庚申辛酉壬亥癸子）+ 天盘加临生成四课（一→四课按递推关系）
- **三传算法**：实现简化版九宗门
  - 贼克（元首 / 重审）：唯一克者取之
  - 比用：多克中取与日干同阴阳者
  - 涉害：都比和或都不比和时取涉害深者
  - 别责：无克时取一课上神为初传（兜底）
- **月将加占时**：按 12 中气取月将（大寒后子、雨水后亥、春分后戌…），公历月份兜底
- **天盘偏移**：`shift = (timeZhi - yueJiang) mod 12`，地盘 d 位对应天盘 (d+shift) % 12 位
- **十二天将**：贵人定位（甲戊庚丑未、乙己子申、丙丁亥酉、壬癸卯巳、辛午寅），昼贵顺布夜贵逆布
- **昼/夜判断**：卯辰巳午未申时为昼（5-17 点），酉戌亥子丑寅时为夜
- **天盘环形图**：CustomPainter 绘制 4 层同心圆（外圈/地盘/天盘/内圈贵人），12 等分放射线，外圈地盘固定 + 内圈天盘旋转 + 中心贵人位
- **四课表**：横向 4 列布局，每课显示「N课」+ 上神（金色）+ 分隔线 + 下神（白色）
- **三传表**：横向 3 列布局，每传显示「初/中/末传」+ 神（大字 22px）+ 天将
- **天将加临表**：12 个 chip 展示「子位:天将」对应关系
- **顶部信息**：农历 + 日时干支 + 月将 + 宗门 + 贵人类型三 chip

### 风水罗盘（luopan · sortOrder 9）

- **磁力计订阅**：sensors_plus 6.1.2，`magnetometerEventStream(samplingPeriod: uiInterval)` 每帧取读数
- **方位角计算**：`atan2(y, x) * 180 / π`，未做倾角补偿（建议用户保持设备水平）
- **24 山罗盘**：CustomPainter 绘制 4 层同心圆 + 24 等分线
  - 外层 24 山：壬子癸丑艮寅甲卯乙辰巽巳丙午丁未坤申庚酉辛戌乾亥
  - 内层八卦：乾坎艮震巽离坤兑（按 8 方位标号）
  - 四正位（子午卯酉）金色高亮 + 四隅位（艮巽坤乾）朱色高亮
  - 顶部红色三角指针固定指北
  - 中心太极点（金色 + 暗色内点）
- **方位读数**：实时显示方位角（1 位小数）+ 坐山（24 山之一）+ 方位（北/东北/东…）
- **平台判断**：仅 Android 显示磁力计罗盘（`PlatformInfo.isAndroid`），桌面端显示占位说明（含 `Icons.explore` 图标 + 平台名称）
- **磁力计生命周期**：组件 dispose 时自动取消订阅，避免内存泄漏
- **开始/停止按钮**：用户可控开关，避免后台消耗

## 技术要点

- 新术注册：`divination_registry.dart` 列表追加 `DaliurenTech(), LuopanTech(),` 两行
- 首页 grid 自动出现新卡片（visibleTechsProvider 按 sortOrder 排序）
- TextPainter 在 _TianPanPainter / _LuopanPainter 中显式 dispose()（遵循硬约束）
- 大六壬算法纯函数式：`divine(year, month, day, hour)` 输入确定即输出确定
- 风水罗盘用 Riverpod Notifier 管理磁力计订阅状态
- 共享 `shan24` 数组给大六壬算法（方位换算）和罗盘 UI（绘制）使用
- sensors_plus ^6.0.0 兼容当前 Flutter 3.11.4（7.x 需 AGP 8.12.1+）
- 历史记录：HistoryStore.add 追加 daliuren 结果（id=daliuren, techName=大六壬, summary=宗门+初传）
- flutter analyze 0 issue
