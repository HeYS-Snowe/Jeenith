## 概述

v2.9.1 续六爻断法深化——在 v2.9.0（旬空/六冲六合）基础上新增**飞神伏神**、**月破日冲暗动**、**三合三刑**三项断法，并补全**六爻仪式入场动画**（铜钱抛落）。至此六爻断法体系完整。`flutter analyze` 0 issue，15 项单测全过。

## 新增

### 飞神伏神
- 用神不上卦时，从本宫八纯卦寻伏神（同位爻为飞神）→ 断辞提示伏藏位置

### 旺衰细分（月破/日冲/暗动）
- 月破：用神冲月建，月内无力 + score 减分
- 日冲：静爻被日冲为暗动（虽静犹动）；动爻被日冲为冲伤反复

### 三合局 / 三刑
- 三合：卦六爻含申子辰/亥卯未/寅午戌/巳酉丑 → 三合局断辞
- 三刑：寅巳申/丑戌未/子卯 + 辰午酉亥自刑 → 刑伤断辞

### 六爻仪式入场动画
- 首页点六爻卡片 → 铜钱抛落动画（复用 ZhouyiRitual，watermark「六爻」）→ 进六爻页
