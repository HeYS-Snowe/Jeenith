# v1.2.0 — 卦辞爻辞数据与文本展示（周易/梅花）

**版本**：1.2.0+9
**状态**：feature（多阶段计划第 1 阶段 · 内容深化）
**构建日期**：2026-07-13

## 概述

为周易 64 卦 + 384 爻建立完整的古文与白话注解数据资产，在周易（金钱卦）与梅花易数结果页展示卦辞与变爻爻辞。这是志极多阶段实施计划的第 1 阶段（共 6 阶段），聚焦内容深化。

## 新增功能

### 卦辞爻辞数据资产

- 新增 `mobile/assets/yijing/hexagrams.json`，包含 64 卦完整数据
- 每卦含卦辞原文（guaCi）+ 白话注解（guaCiNote）+ 爻辞数组（yao）
- 每爻含爻位称名（初九/六二/上六等）+ 爻辞原文（ci）+ 白话注解（note）
- 乾坤两卦各 7 爻（含用九/用六），其余 62 卦各 6 爻
- 共 64 条卦辞 + 386 条爻辞（含用九用六）
- 以《周易正义》为底本逐卦校对

### HexagramTexts 加载器

- 新增 `mobile/lib/data/yijing/hexagram_texts.dart`
- 启动时从 rootBundle 预加载到内存 Map，之后 O(1) 同步查询
- 提供 guaCi / guaCiNote / yaoCi / yaoCiNote / posName / yongCi 查询接口
- posName 按"初九/六二/上六"规则生成爻位称名（阳爻称九、阴爻称六，初位与上位位置在前）

### 周易（金钱卦）文本展示

- 起卦结果下方追加"本卦卦辞"卡片（原文 + 白话注解）
- 变爻追加"变爻 · 初九"等爻辞卡片（每个变爻一张）
- 有变卦时追加"之卦卦辞"卡片
- 复制结果文本包含全部卦辞爻辞段落

### 梅花易数文本展示

- MeihuaResult 新增 dongPos / dongYang getter（从经卦二进制值提取动爻阴阳）
- 起卦结果下方追加"本卦卦辞"/"动爻爻辞"/"之卦卦辞"三张卡片
- 梅花动爻只一爻，展示该爻的爻辞原文与注解
- 复制结果文本包含全部卦辞爻辞段落

## 技术要点

- `divine()` 保持纯算法不读文本，UI 层查询 HexagramTexts 单例
- 爻位称名规则：阳爻称"九"，阴爻称"六"，自下而上为初/二/三/四/五/上；初位与上位把位置放前（初九、上六），其余把阴阳放前（九二、六三）
- 数据加载在 main() 启动阶段完成，确保结果页同步访问无延迟
- flutter analyze 0 issue
