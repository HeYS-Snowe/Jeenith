# v1.3.0 — 紫微斗数 v2 全套星曜安星

**版本**：1.3.0+10
**状态**：feature（多阶段计划第 2 阶段 · 内容深化）
**构建日期**：2026-07-13

## 概述

为紫微斗数实现完整的星曜排盘：14 主星 + 六吉六煞 + 博士十二神 + 神煞（天马/华盖/桃花/红鸾/天喜），并在结果页展示环形命盘图与宫位详情。这是志极多阶段实施计划的第 2 阶段（共 6 阶段），聚焦紫微斗数的内容深化。

## 新增功能

### 安星算法核心

- 新增 `mobile/lib/features/ziwei/algorithm/star_placement.dart`
- 实现 `StarPlacement` 与 `StarChart` 数据结构（按 12 宫地支索引）
- 实现 `placeStars()` 函数，输入年干/年支/月/日/时/五行局数，输出完整星盘
- 算法涵盖：
  - 紫微星定位：`ziweiPos = (2 + (day-1) ~/ ju) % 12`
  - 天府星对称定位：`tianfuPos = (4 - ziweiPos + 12) % 12`（关于寅申轴对称）
  - 紫微系 6 星（紫微/天机/天同/太阳/武曲/廉贞）
  - 天府系 8 星（天府/太阴/贪狼/巨门/天相/天梁/七杀/破军）
  - 六吉星：左辅/右弼（月）、文昌/文曲（时）、天魁/天钺（年干查表）
  - 六煞星：擎羊/陀罗（年干查表）、火星/铃星（年支三合起 + 时支）、地空/地劫（时支）
  - 博士十二神：从年干查起始宫位，顺布 12 宫
  - 神煞：天马/华盖/桃花（年支三合查表）、红鸾/天喜（年支算）
- 每宫星曜按类别排序：主→吉→煞→博→神

### 星曜常量表

- 新增 `mobile/lib/features/ziwei/data/stars.dart`
- 14 主星名、紫微系/天府系偏移表、六吉六煞名
- 博士十二神名序 + 起始位表
- 天魁天钺/擎羊陀罗/火星铃星/天马华盖桃花的年干年支定位表
- StarCategory 枚举与 starCategory 映射

### 环形命盘绘制

- 新增 `mobile/lib/features/ziwei/ui/star_chart_painter.dart`
- CustomPainter 实现：12 宫扇形（每宫 30°），子位于底部，逆时针布列
- 命宫金色高亮、身宫蓝色高亮（描边）
- 每宫扇形内：地支字 + 宫名 + 主星名 + 辅星缩略
- 中央圆：太极点 + 命宫干支 + 五行局
- 颜色规则：主星金、吉星绿、煞星红、博士神灰、神煞紫
- TextPainter 显式 dispose（遵守项目硬约束）

### 紫微斗数页面升级

- 删除 v1 占位文字
- 4×4 GridView 替换为 CustomPaint + StarChartPainter（高度 380）
- 命盘下方追加"宫位详情"列表：12 行，每行显示地支+宫名+主星+吉星+煞星+博士神+神煞
- 复制结果文本包含各宫星曜完整列表

## 技术要点

- `divine()` 在 v1 基础上扩展，新增 `final StarChart stars` 字段
- 安星算法是确定性函数（无随机），输入生辰参数即可计算
- 年支索引通过 `_dizhi.indexOf(ec.getYear()[1])` 获取
- 五行局数通过 `_juNum` 映射（金4/木3/水2/火6/土5）
- StarChart 实现 `==` 与 `hashCode` 用于 shouldRepaint 优化
- flutter analyze 0 issue
